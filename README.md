# DevLab Website

This is the DevLab website using React, TypeScript, and Vite.

## Tech Stack and Libraries

The technology stack for the website is React, TypeScript, and Vite.

We use Vite as a bundler to render static files. Run `npm run build` to compile the site.

TypeScript is in strict mode with strict JSX.

`react-router-dom` directs pages within the site. Routing-related files are in [src/router](src/router).

## Contributing

We are currently working on setting up the base site and design.

When making a new page, create a new branch for the page called `feature/page-name`. For example, the home page branch is called `feature/home-page`.

Commit your work to your branch as necessary and push it to the repository. When the page is completed, create a pull request and describe your changes. If it is accepted, you'll need to pull the new code into main on your local machine. If not, you'll need to modify your branch, push it again, and make a new pull request with the issues fixed.

We're still thinking about the layout and design of the website. If you have any suggestions for how it should look or any other helpful resources or information, make an issue. Thanks!
